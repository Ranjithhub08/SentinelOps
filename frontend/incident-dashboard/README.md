# Incident Dashboard

## Overview
The Incident Dashboard is the primary web interface for SentinelOps. It provides a real-time, interactive, and visually premium visualization of system metrics, active anomalies, incident tracking, and AI-generated root cause explanations.

Built utilizing **Next.js 14** (App Router) and **Tailwind CSS**, it heavily leverages modern styling techniques like glassmorphism and gradient typing to deliver an exceptional user experience for Site Reliability Engineers and DevOps personnel.

## Key Screens
1. **Overview**: An aggregation of active incidents, global system health metrics, and a cascading alert feed.
2. **Incidents**: A focused, interactive tabular view of all anomalies promoted to incident status. Selecting an incident reveals a detailed side panel.
3. **Alert Feed**: A dedicated historical log of all the real-time alerts generated by the backend alert service.
4. **AI Insights**: A contextual hub where the system provides macroscopic recommendations based on observed behavioral clusters across the infrastructure.

## API Integration Architecture
The dashboard connects dynamically to the underlying microservices suite:
- Incident data is polled from the **Incident Management Service**:
  - `GET http://localhost:8004/incidents`
  - `GET http://localhost:8004/incidents/{incident_id}`
- For the purposes of presenting a unified AI-enriched view without requiring a complex GraphQL gateway upfront, the UI gracefully bridges missing data using visual cues and overlays when deep root-cause integrations are pending.

## Running Locally

### Prerequisites
- Node.js 18+
- Backend services running (especially `incident-management-service` on port 8004).

### Setup
From the `frontend/incident-dashboard` directory:
```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

The unified dashboard will be accessible at [http://localhost:3000](http://localhost:3000).

## Project Structure
- `src/app/`: Next.js pages representing the distinct views (Overview, incidents, alerts, insights).
- `src/components/`: Reusable, atomic UI elements (React Components).
- `src/app/globals.css`: Core Tailwind definitions and custom glassmorphism utilities matching the SentinelOps dark-mode theme.
